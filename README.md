# Mentor-Mentee Platform (Android)

An Android mobile application designed to bridge the gap between mentors and mentees, facilitating seamless communication, guidance, and progress tracking. 

## 🚀 Key Features
* **Role-Based Access:** Distinct user interfaces and capabilities for 'Mentors' and 'Mentees'.
* **Real-Time Communication:** Integrated messaging to allow instant communication between paired users.
* **Profile Management:** Users can update their skills, goals, and professional backgrounds.
* **Secure Authentication:** User login and registration handled securely.
